# Description
Creates a Vector2 with the given x and y arguments. If no arguments are given the Vector2 is assumed to be 0, 0.

# Example
``` c++
Vector2 v1 = Vector2(10,10);
Vector2 v2;
Vector2 v3 = Vector2(0,0); // Same as v2
```
