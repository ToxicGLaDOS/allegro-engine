# Description
Calculates the dot product of two vectors. The dot product is equal to the sum of the components of the product of the two vectors.

# Examples
``` c++
Vector2 v1 = Vector2(1, 2);
Vecotr2 v2 = Vector2(3, 4);
v1.dot(v2); // Equal to 1*3 + 2*4
```
